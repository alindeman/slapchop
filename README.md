# Slapchop

  Slap your troubles away with slap chop

## Installation

Add this line to your application's Gemfile:

    gem 'slapchop'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install slapchop

## Usage

```ruby
object.slap_chop(:onions)
```
